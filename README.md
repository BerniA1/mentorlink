# MentorLink

Plataforma web que conecta mentores y mentees dentro de la comunidad universitaria de la UPV (Universitat Politècnica de València).

## Descripción

MentorLink facilita el emparejamiento entre estudiantes que buscan orientación (mentees) y estudiantes o egresados con experiencia que desean compartirla (mentores). La plataforma cubre tanto mentorías académicas (asignaturas, proyectos) como profesionales (prácticas, salida laboral).
